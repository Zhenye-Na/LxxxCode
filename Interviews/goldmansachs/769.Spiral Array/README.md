# 769. Spiral Array

**Description**

Given an integer `n`, return a spiral array of `n * n` sizes.

**Example**

Example 1:

```
Input : n = 3
Output : 
[
[1,2,3]
[8,9,4]
[7,6,5]
]
```

Example 2:

```
Input: n = 5
Output: 
[
 [1,2,3,4,5]
 [16,17,18,19,6]
 [15,24,25,20,7]
 [14,23,22,21,8]
 [13,12,11,10,9]
]
```
