# 1197. Find Bottom Left Tree Value

**Description**

Given a binary tree, find the `leftmost` value in the `last` row of the tree.

You may assume the tree (i.e., the given root node) is not `NULL`.

**Example**

Example 1:

```
Input: {2,1,3}
Output: 1

Explanation:
  2
 /  \
1   3
```

Example 2:

```
Input: {1,2,3,4,5,6,#,#,7}
Output: 7

Explanation:
          1
        /  \
      2     3
    /  \    / 
  4     5 6
   \
    7
```
